Change the config file at crazyswarm2_ws/src/crazyswarm2/crazyflie/config/crazyflies.yaml

To launch the ROS nodes to connect with the Crazyflie:

source crazyswarm2_ws/install/setup.zsh
ros2 launch crazyflie launch.py backend:=cflib

# use cflib backend if connecting to CrazySim

To launch an example script:

ros2 run crazyflie_examples hello_world
